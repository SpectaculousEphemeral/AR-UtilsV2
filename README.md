# AR-UtilsV2
A Utility Module for Arsenal.

# How To Use
```lua
-- Import AR-UtilsV2
local utils = loadstring(game:HttpGet("https://raw.githubusercontent.com/Storm99999/AR-UtilsV2/main/src/utils.lua"))()
-- Call our disable function
utils.DisableAC()
-- Let BH execute after a second.
task.wait(1)
utils.ExecuteBH()```
